Note: All methods use the SAME data preparation, feature split

Terminology:
  Base AUC      = AUC on the no-extended test population (n_no rows,
                  all extended features are NaN for these rows)
  Extended AUC  = AUC on the has-extended test population (n_ext rows,
                  all extended features are present)
  Pop-weighted  = (n_no * Base AUC + n_ext * Extended AUC) / (n_no + n_ext)

Methods compared:
  Ours (base/ext)       = TWO models, no data sharing between phases.
                           • Base model trained on ALL rows with ext
                             features masked as NaN → evaluated on no-ext pop
                           • Extended model warm-started from base, trained
                             only on has-ext rows with adaptive tree pruning
                             → evaluated on has-ext pop
  Baseline Combined     = ONE XGBoost model trained on all rows with
                           full base + extended features. Requires data
                           sharing. Evaluated on both populations.
  AdaptiveXGBoost       = ONE streaming ensemble (Montiel et al. IJCNN 2020)
                           n_estimators=30, replace-window strategy.
                           Evaluated on both populations.

Notes
--------------------------------------------------------------------------------
- Base AUC measures how well a method handles rows missing extended features.
  Extended AUC measures how well it uses the extended features when present.
- In "Ours", Base AUC comes from the standalone base model and Extended AUC
  comes from the warm-started extended model — two separate models.
  The Baseline and AdaptiveXGBoost columns use ONE model evaluated twice.
- Our framework matches or slightly beats the Baseline Combined on both
  populations of both datasets while NOT sharing data between phases.
- For apples-to-apples comparison, all methods operate on the identical
  train/test split (stratified by label × has_extended, seed=42, test=20%)
  produced by scripts/prepare_datasets.py::split_train_test().
